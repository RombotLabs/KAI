# KAI
KAI is a DSGVO-friendly ai chat platform designed for schools.

## Kurze Info an alle Devs
Bitte immer, wenn ihr ein neues Modul nutzt und über pip installiert, hängt bitte den Modulnamen ans Ende von requirements.txt ran.

### Mehr infos
ufm.py ist eine Bibliothek, die für das Management von User-Chats und hochgeladenen dateien zuständig ist.
Bei einem neuen User wird z.B. ein neues Verzeichnis mit allen chats erstellt.
